git과 git-hub

깃허브에 파일을 올리는 것 >> 호스팅

넷리파이 >> 도메인을 달아주는 역할을 하는 사이트



git config --global user.name "유저네임"
git config --global user.email "유저이메일"

로그인을 하게 되는 것

git add.

.이 의미하는 것은 모든 파일을 다 올려버리겠다

git status

어떠한 파일이 추가되었고 또한 어떠한 파일이 수정되었는지 모두 확인이 이루어집니다.

git remote add origin 주소

git commit -m "커밋이름"

git push origin master

rm -rf git.

master에서 잠깐 벗어나게 됩니다.

git init

작업을 반복한다

git clone 경로주소 "폴더명"

freshman  폴더가 생겼다고 가정

cd freshman

code.
비쥬얼 스튜디오 코드가 열리면서 해당파일을 오픈한다

git push origin freshman
